# API Contract

## UserController
### POST /register

**Request:**
- Method: POST
- Path: /register
- Headers: Content-Type: application/json
- Body: 
  ```json
  {
    "email": "string",
    "password": "string",
    "name": "string"
  }
  ```

**Responses:**
- 200 OK
  - Content-Type: application/json
  - Body:
    ```json
    {
      "status": "success",
      "timestamp": "timestamp",
      "data": {
        "id": "string",
        "name": "string",
        "email": "string",
        "role": "string"
      }
    }
    ```
- 400 Bad Request
  - Content-Type: application/json
  - Body:
    ```json
    {
      "status": "error",
      "timestamp": "timestamp",
      "message": "string"
    }
    ```
